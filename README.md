# Suryamitr Connect

Suryamitr Connect is an open, non-profit-oriented digital initiative focused on sustainability, ESG awareness, and community-driven impact. The project exists to share transparent information, practical resources, and simple digital tools that support environmental responsibility, social good, and ethical governance.

## Purpose

The purpose of this repository is to:

* Share ESG and sustainability-focused content openly
* Enable learning and collaboration around social impact
* Demonstrate low-cost, transparent digital infrastructure for public-interest initiatives

This project is intentionally lightweight, static, and fully auditable.

## Website

The project is hosted using GitHub Pages.

Live website:

## Technology Stack

* Static HTML and CSS
* GitHub Pages hosting
* Privacy-first traffic analytics (GoatCounter)
* No cookies, no user tracking, no personal data collection

## Analytics and Privacy

Traffic analytics are implemented using GoatCounter:

* No cookies
* No fingerprinting
* No personal or identifiable data collection
* Designed to be GDPR-friendly

## Governance Principles

* Transparency over growth
* Ethics over engagement metrics
* Open access over monetization pressure
* Long-term social value focus
